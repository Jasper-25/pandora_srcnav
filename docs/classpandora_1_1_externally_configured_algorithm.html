Detailed Description

Externally configured algorithm class. Provides ability for external specification of algorithm-defined parameter blocks. A single call to GetExternalParameters per algorithm type, per Pandora instance is enforced to prevent misuse. In the case of multiple instances of a given algorithm type, must revert to standard xml-based configuration.
